## Phishing URL & Screenshot Detector

This is a Streamlit web app that checks if a website is **phishing or legitimate**.

### Features

- Paste a **URL** to detect phishing
- Upload a **screenshot** of a webpage to get a prediction
- Shows **prediction result** and **confidence percentage**
- Easy to run, made for learning/demo use

### How to Run

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
